Система рекомендаций фильмов с двухконтурной архитектурой (офлайн/онлайн), построенная на основе коллаборативной и контентной фильтрации.

## Архитектура системы

Система построена на двух основных контурах:

### Офлайн-контур (ресурсоёмкие фоновые задачи)


1. **Модуль сбора и предобработки данных** (`DataPipeline`)
   - Извлечение данных из PostgreSQL
   - Нормализация и векторизация признаков
   - Создание TF-IDF матриц и векторов жанров/актеров/режиссеров
   - Агрегация данных для обучения

2. **Модуль обучения и валидации моделей** (`ModelTrainer`)
   - Асинхронное периодическое переобучение
   - Обучение SVD, NMF, ALS моделей
   - Построение моделей предсказания оценок и ранжирования
   - Сохранение результатов (матрицы, векторы признаков)

3. **Промежуточный кэширующий слой** (`CacheManager`)
   - Хранение предвычисленных топ-N рекомендаций в Redis
   - Ускорение работы за счет кэширования
   - Решение проблемы согласованности данных

### Онлайн-контур (реальные рекомендации с низкой задержкой)

1. **Получение контекста** (`ContextHandler`)
   - ID пользователя, история, параметры запроса
   - Стратегии холодного старта для новых пользователей

2. **Генерация кандидатов** (`CandidateGenerator`)
   - Объединение результатов коллаборативной и контентной фильтрации
   - Дедупликация кандидатов

3. **Ранжирование кандидатов** (`Ranker`)
   - Упорядочивание по предсказанной полезности
   - Учет разнообразия, недавности и контекста

4. **Постобработка и фильтрация** (`Postprocessor`)
   - Исключение уже просмотренного
   - Учет ограничений и чередование жанров
   - Добавление объяснений

5. **Модуль логирования и обратной связи** (`FeedbackLogger`)
   - Фиксация действий пользователя
   - Асинхронная запись в файлы и БД
   - Данные для переобучения моделей

## Функциональные возможности

- **Персонализированные рекомендации** на основе истории оценок и отзывов пользователя
- **Гибридные рекомендации** (коллаборативные + контентные)
- **Поиск фильмов** по названию (русскому/английскому)
- **Каталог с фильтрацией** по жанрам, году, стране, актерам, режиссерам
- **Страница просмотренных фильмов** с историей оценок и отзывов
- **Статистика пользователя** (распределение по жанрам, годам, оценкам)
- **Русскоязычный интерфейс** с поддержкой русских названий жанров, актеров, режиссеров
- **Кэширование рекомендаций** для быстрого ответа
- **Асинхронное логирование** обратной связи

## Модели рекомендаций

### Используемые алгоритмы

1. **User-based Collaborative Filtering**
   - Находит похожих пользователей
   - Рекомендует фильмы, которые понравились похожим пользователям

2. **Content-based Filtering**
   - Использует TF-IDF для текстовых признаков
   - Учитывает жанры, актеров, режиссеров, сюжет

3. **SVD (Singular Value Decomposition)**
   - Матричная факторизация
   - Выявляет скрытые паттерны в предпочтениях

4. **ALS (Alternating Least Squares)**
   - Оптимизирован для implicit feedback
   - Быстрая работа с большими матрицами

5. **Гибридный подход**
   - Комбинирует все методы с настраиваемыми весами
   - Учитывает популярность, свежесть и разнообразие

## Технологический стек

### Backend
- **Python 3.9+**
- **Flask** - веб-фреймворк
- **PostgreSQL** - основная база данных
- **Redis** - кэширование рекомендаций
- **SQLAlchemy** - ORM

### Машинное обучение
- **scikit-learn** - TF-IDF, SVD, NMF, метрики
- **implicit** - ALS для implicit feedback
- **NumPy/Pandas** - обработка данных
- **scipy** - разреженные матрицы

### Frontend
- **Bootstrap 5** - UI компоненты
- **Chart.js** - визуализация статистики
- **Font Awesome** - иконки
